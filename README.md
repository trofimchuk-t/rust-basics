## Installation
https://www.rust-lang.org/tools/install

```bash
# Install Rust:
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Check vesrions:
rustup --version
rustc --version

# Update Rust:
rustup update
```
